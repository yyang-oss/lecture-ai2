# API利用ノート

## 使用モデル
- **モデル名**: `gemini-3-flash`
- **エンドポイント**: `https://generativelanguage.googleapis.com/v1/models/`

## 制限・仕様
- **無料枠**: 1分あたり15リクエスト / 1日1500リクエスト（2026年3月現在）
- **対応形式**: mp3, wav, m4a, ogg等
- **最大入力**: 1回あたり最大約1時間程度の音声まで（Flashモデルのコンテキスト制限内）

## プロンプト戦略
タグベースのセグメンテーション（`[TRANSCRIPT]`, `[SUMMARY]`, `[POINTS]`）を用いて、生成AIの出力をプログラムで確実にパースする手法を採用。
